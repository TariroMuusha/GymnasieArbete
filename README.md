# Vilket av två kontrollsätt är mest effektivt för att kontrollera en radiostyrd bil?
Här finner du koden som använts i ett gymnasie arbete, koden styr en rc bil med hjälp av en telefon kopplad via en webserver till en arduino eller rörelse sensor kopplad direct in i arduinon. 

Gymnasiearbete läsåret 2020-2021 av Emil Eriksson, Isac Holm, Maarten Coelingh & Tariro Muusha.
